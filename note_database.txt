- (보이지 않는 곳에서 묵묵히 동작하고 있는) 정보기술의 심장 = 데이터베이스

- 데이터 가공 -> 다양한일 (ex 인터넷에 연결된 웹과 앱을 통해 소식 지식 전파, 빅데이터 인공지능과 같은 기술을 이용해서 대규모의 데이터로부터 통찰력 있는 분석결과를 뽑아낼수도 있음)

- 파일 : 성능, 보안, 편의성에 한계 (데이터를 저장하는 가장 간단한 방법)

- 상상력 -> 크고 복잡하고 위험한 데이터!
데이터베이스의 복잡성에 대해서 공감!
공감 없는 이해는 모래위에 지어진 성!

-입력 출력

- description : 본문
sort : 정렬

- 구조적으로 데이터 저장(정리정돈) -> 데이터를 가공하는 것이 훨씬 더 쉬워짐

- File < Spreadsheet(ex excel) < Datebase

- 프로그램적으로, 컴퓨터 언어를 이용해서 데이터 CRUD 할수있다
= 데이터베이스 장점 : 자동화 할 수 있다

- 데이터베이스 시장의 절대강자 : 관계형 데이터베이스 (Relational dBMS)
Oracle, MySQL - 관계형 데이터베이스
MongoDB - 다른것
